# ⌚ Watch E-Commerce Website

## 📖 Project Overview

This project is a **Watch E-Commerce Website** developed as part of my **third-year engineering project**.
The website allows users to browse different watches, view product details, and interact with features such as cart, wishlist, and product reviews.

The main goal of this project was to learn **front-end web development and user interface design for e-commerce platforms**.

---

## 🚀 Features

* 🛍 Product listing for different watches
* ❤️ Wishlist functionality
* 🛒 Add to cart system
* ⭐ Product reviews section
* 🔎 Related products suggestion
* 📱 Responsive design for multiple devices

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Interactivity and dynamic functionality

---


## 🎯 Learning Outcomes

Through this project, I learned:

* Designing **e-commerce website layouts**
* Implementing **cart and wishlist functionality**
* Creating **responsive web pages**
* Improving **JavaScript interaction and UI design**
